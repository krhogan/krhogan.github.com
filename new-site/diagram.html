    • 100 Things Diagram

    28"x73"

    This project involved gathering a simple data set and defining methods to structure that data. I organized my data by 7 attributes: the category Stumble-Upon put it in, type of media (images, video, text, interactive), the date (day, month, year), if any special occasion was happening (birthdays, school breaks, etc.), how often I returned, if I'm still interested or not, and my level of usage for that day.
